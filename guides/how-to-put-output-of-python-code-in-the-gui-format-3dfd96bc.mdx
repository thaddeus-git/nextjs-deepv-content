# How to put output of python code in the GUI format?: 1 Method + Performance Guide

## Quick Answer

To display the output of your Python code in a Graphical User Interface (GUI) window instead of the console, the most straightforward and widely adopted method is to use the `tkinter` library. You can achieve this by creating a `tkinter` window, adding a `Label` widget, and dynamically updating its `textvariable` with the output from your backend Python logic.

```python
import tkinter as tk

def get_backend_output():
    """Simulates a backend Python function generating output."""
    # Replace this with your actual Python code logic
    result = "Hello from Python Backend!"
    return result

def update_gui_output():
    """Fetches backend output and updates the GUI label."""
    output_text = get_backend_output()
    output_var.set(output_text)

# Initialize the main window
root = tk.Tk()
root.title("Python GUI Output")
root.geometry("400x200")

# Create a StringVar to hold the text for the Label
output_var = tk.StringVar()
output_var.set("Click the button to get output...")

# Create a Label widget to display the output
output_label = tk.Label(root, textvariable=output_var, font=("Arial", 14), wraplength=350)
output_label.pack(pady=20)

# Create a Button to trigger the update
update_button = tk.Button(root, text="Get Output", command=update_gui_output, font=("Arial", 12))
update_button.pack(pady=10)

# Start the Tkinter event loop
root.mainloop()
```
